ArchiTech's <b>Zedboard</b> board is a deeply embedded board.<br>
It's best feature is its capability to run Linux using only the internal ram.
For greater memory requirements, the board can mount external RAM memory and the processor
can use both the internal and the external memories, in which case the internal memory will be used by the frame buffer driver.
